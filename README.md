# Ecommerce Platform

A monorepo-based ecommerce platform with:
- Web storefront
- Admin dashboard
- Mobile app
- Shared packages for UI, database, API, types, and payments

## Tech Stack
- Turborepo
- Next.js
- Expo / React Native
- PostgreSQL
- Prisma
- TypeScript
